## Execução

1. **Construa o Projeto:**
   ```bash
   mvn clean install
   ```

2. **Execute o Projeto:**
   ```bash
   java -jar target/nome-do-projeto.jar
   ```

3. **Acesse a Aplicação:**
   - Acesse a aplicação em [http://localhost:8080](http://localhost:8080).
